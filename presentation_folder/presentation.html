Anomaly detection with autoencoders

  • Autoencoder: two networks
    • Encoder \(\phi_E\): maps into latent space
    • Decoder \(\phi_D\): maps back
    • AE trained to reconstruct normal data well, i.e., minimize \(d(z,\phi_D(\phi_E(z))\)
  • Thus: anomalous time series should reconstruct poorly

Autoencoders: a bit more

  • Just linear networks! Works well enough for our purposes.
  • Trained by comparing decoder reconstruction to input.
  • Seems to learn large-scale features of data...

How to actually detect?

\{d(z,\phi_D(\phi_E(z))), \forall z \in D \}\\ \text{ Reconstruction losses on training data }
1-\operatorname{CDF}_{\pi_D}(d(x)) = 1 -\mathbb{P}_{\pi_D}(d(z) \leq d(x)) = \mathbb{P}_{\pi_D}(d(z) > d(x))\\ \text{ Anomaly score }
d(\cdot ) \coloneqq d(\cdot,\phi_D(\phi_E(\cdot)))\\ \text{ Abbreviation for training loss }

Scalability through an ensemble

  • Primary objective: Utilize an ensemble when the dataset exceeds the capacity of a single machine
  • Enables the application of a distributed learning framework:
    • Potential acceleration of training
    • Potentially respecting privacy data
  • Low transfer costs by outputting metrics (anomaly score)

Ensemble outline

  • Data set \(D\), size \(S_D\)
  • Each node (machine) can handle at most \(S_1\) datapoints within a reasonable timeframe
  • Use ensemble if \(S_D > S_1\), otherwise use a single machine
  • Partition data into \(N\) chunks: \(D = \cup_{i=1}^N D_i\), where \(|D_i| < S_1\)
  • Use \(N\) nodes
  • Fault tolerence: The system continues to function (for inference) effectively even if one or more nodes fail.
  • Dynamic scaling: As datasets grow, additional nodes can be incorporated to maintain performance

Ensemble procedure

The process proceeds by letting:

  1. Training:
    1. Node \(i\) receives data \(D_i\), \(i = 1,\ldots N \)
    2. Node \(i\) defines a model \(m_i\), \(i = 1,\ldots N \)
    3. Model \(m_i\) trained on \(D_i\) until criteria is met, \(i = 1,\ldots N \)
  2.  Inference:
    1. The master node is fed a set \(D_{\operatorname{test}}\) to perform the task
    2. Assuming \(|D_{\operatorname{test}}|<S_1\), send whole set to each node \(i\)
    3. Node \(i\) evaluates and returns \(1-\operatorname{CDF}_{\pi_D}(d_i(D_{\operatorname{test}}))\)
    4. Master node computes the mean:

 

 

\frac{1}{N}\sum_{i=1}^N \left(1-\operatorname{CDF}_{\pi_D}(d_i(D_{\operatorname{test}}))\right)
